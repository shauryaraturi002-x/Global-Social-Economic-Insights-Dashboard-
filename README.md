# Global-Social-Economic-Insights-Dashboard-
🚀 Global Social & Economic Insights Dashboard | Power BI Project

Excited to share my latest end-to-end data analytics project where I built a Global Social & Economic Insights Dashboard from scratch!

🔹 What I did:

Performed data cleaning & transformation using Power Query

Designed a structured data model with relationships

Created impactful DAX measures (AVG Happiness, Poverty %, GDP per Capita, Population, Health Index, etc.)

Built an interactive, insight-driven Power BI dashboard

📊 Dashboard Highlights:

🌍 Global Overview

Avg Happiness Score

Avg Poverty %

Total Population

Poverty levels across countries

Map visualization of Happiness Score by Country

📈 Social Development Analysis

Education Index by Happiness Category

Health Index vs Happiness Score

Regional Happiness distribution

GDP per Capita vs Happiness Score correlation

💰 Economic Analysis

GDP Distribution by Region

GDP vs Happiness relationship

Unemployment Rate comparison across regions

🔎 Key Insights:

Strong correlation between GDP per capita & Happiness Score

Regions with higher Health & Education Index tend to report higher happiness

Noticeable regional differences in poverty & unemployment patterns
